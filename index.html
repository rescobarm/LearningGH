Agregando cambio !!

  • git init (Inicializa el repositorio)
  • git status (Status de los archivos)
  • git add -A ó git add filename (Agrega todos los archivos al stage área)
  • git rm --cached .gitignore (Saca de la lista del stage área)
  • git commit -m "Mensaje" (Agrega archvis al repositorio los archivos)
  • git log (Muestra commits realizados)
  • gti diff (Muestra las diferencias que hay entre lo que estaba y los cambios realizados)

Comandos linux

  • ls -alh (Muestra ficheros y archivos a detalle)
