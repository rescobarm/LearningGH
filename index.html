Agregando cambio !!

  • git init (Inicializa el repositorio)
  • git status (Status de los archivos)
  • git add -A ó git add filename (Agrega todos los archivos al stage área)
  • git rm --cached .gitignore (Saca de la lista del stage área)
  • git commit -m "Mensaje" (Agrega archvis al repositorio los archivos)
  • git log (Muestra commits realizados)
  • git diff (Muestra las diferencias que hay entre lo que estaba y los cambios realizados)
  • git pull origin master (Descargamos la última versión del códio del proyecto)

WORK FLOW!!! branch

  • git branch [branch name] (Crear rama para modificaciones)
  • get checkout [branch name] (Cambiar a la nueva rama)
  • git add -A (Agrega los cambios realizados de la rama)
  • git commit -m "Confirmando cambios realizados en la nueva rama"
  • git push -u origin [branch name]
  • git branch -a (Muestra todas las branch, remotas y locales)
